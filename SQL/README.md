# Contenidos
La carpeta `...\SQL` contiene los achivos `BodegaJaime.sql` y `Productos de ejemplo.sql`, ambos programados en y para funcionar en `MySQL WORKBENCH 8.0 CE`. Contienen la relación de las tablas así como procedimientos y funciones, y el otro tiene la información de ejemplo que se utilizará.
# Uso de los archivos
## `BodegaJaime.sql`
1. Descargar el archivo y abrirlo con `MySQL`.
2. Asegurarse de tener el _status_ del servidor sea activo
3. Ejecutar el script.
## `Productos de ejemplo.sql`
1. Descargar el archivo y abrirlo con `MySQL`.
2. Asegurarse de tener el _status_ del servidor sea activo
3. Ejecutar el script.
# Uso de la base de datos
Para conectar el _backend_ a la base de datos tan solo necesitas asegurarte que el servidor esté activo.
